Enable Blacklist Mode:

Note: Set the following about:config values to true

  • browser.tabs.loadInBackground
  • browser.tabs.loadBookmarksInBackground
  • browser.tabs.loadDivertedInBackground

Control the load states via. White- or Blacklist mode and RegEx-URL-Matches)

  • Whitelist Mode (unchecked): matched URLs will be allowed to load in the background
  • Blacklist Mode (checked): matched URLs will NOT be allowed to load in the background
  • Example 1: ^https:\/\/www\.twich\.tv.*, matches all twitch.tv sites
  • Example 2: ^https:\/\/www\.youtube\.com.*, matches all youtube.com sites
